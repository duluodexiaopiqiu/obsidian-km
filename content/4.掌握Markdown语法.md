# 4. 掌握 Markdown 语法（Obsidian 的内容基础）

## 为什么要学 Markdown？

Obsidian 的所有笔记都是 Markdown 格式，学会 Markdown 就等于完全掌握了 Obsidian 的内容编辑能力。Markdown 的特点是：

- **简单** —— 几分钟就能学会
- **纯文本** —— 不依赖任何编辑器
- **通用** —— GitHub、飞书、Notion 都支持
- **专注内容** —— 让你专注于写什么，而不是长什么样

## 核心语法速查

### 1. 标题

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

快捷键：`Ctrl+1~6` 快速设置标题级别

### 2. 文本样式

```
**加粗**
*斜体*
***加粗斜体***
~~删除线~~
==高亮==（Obsidian 特有）
`行内代码`
```

### 3. 列表

**无序列表：**
```
- 项目一
- 项目二
  - 子项目（缩进2个空格）
  - 子项目
- 项目三
```

**有序列表：**
```
1. 第一步
2. 第二步
3. 第三步
```

### 4. 任务清单（Checkbox）

```
- [ ] 待办任务
- [x] 已完成任务
- [ ] 未完成任务
```

渲染效果：
- [ ] 待办任务
- [x] 已完成任务
- [ ] 未完成任务

### 5. 引用

```
> 这是一段引用
> 可以多行
>> 引用里还可以嵌套引用
```

### 6. 代码块

````
```javascript
// 语言标识符可选，用于语法高亮
function hello() {
  console.log("Hello Obsidian!");
}
```
````

支持的语言：`javascript`、`python`、`html`、`css`、`java`、`go`、`rust` 等。

### 7. 分割线

```
---
```

### 8. 表格

```
| 姓名 | 年龄 | 职业 |
|------|------|------|
| 张三 | 28   | 工程师 |
| 李四 | 32   | 设计师 |
```

### 9. 图片与链接

```
# 链接
[显示的文本](https://example.com)

# 图片
![图片描述](图片路径.jpg)

# Obsidian 内部链接（核心功能，下节详讲）
[[另一篇笔记]]
[[另一篇笔记|自定义显示文本]]
```

### 10. 注释

```
%%这是注释内容，只在编辑模式下可见%%
```

## 在 Obsidian 中写 Markdown 的小技巧

### 实时预览
Obsidian 默认是"实时预览"模式，你写 Markdown 符号的同时就能看到渲染效果，不用来回切换。

### 快捷键辅助

| 效果 | Windows | macOS |
|------|---------|-------|
| 加粗 | `Ctrl+B` | `Cmd+B` |
| 斜体 | `Ctrl+I` | `Cmd+I` |
| 链接 | `Ctrl+K` | `Cmd+K` |
| 代码 | `` Ctrl+` `` | `` Cmd+` `` |

### 换行技巧
- 同一段落内换行：直接按 `Shift+Enter`
- 新起一个段落：按 `Enter`
- 在 Obsidian 设置中关闭"严格换行"后，单次 `Enter` 也能换行
