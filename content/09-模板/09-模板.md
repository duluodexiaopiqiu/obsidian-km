# 8.0.1. 模板 —— 让笔记录入事半功倍(workbuddy+obsidian结合使用)

## 什么是模板？

模板就是**笔记的预制框架**。每次创建同类型的笔记时，不用从头写起，直接套用模板，填空即可。

举个例子：

- 每天写日记 → 建一个日记模板，包含日期、天气、今日要点
- 读书笔记 → 建一个读书模板，包含书籍信息、核心观点、个人思考
- 会议记录 → 建一个会议模板，包含时间、参会人、议题、决议

## 如何启用模板功能？

Obsidian 的模板功能由**核心插件"模板"**提供，默认已启用。

### 检查是否开启

1. 进入设置（左下角齿轮）
2. 点击"核心插件"
3. 找到"模板"，确认已开启

### 设置模板文件夹

1. 设置 → 核心插件 → 模板
2. 在"模板文件夹位置"中，指定一个文件夹（如 `模板/` 或 `6-模板/`）
3. （可选）设置模板快捷键，默认是 `Ctrl+Shift+T`（macOS: `Cmd+Shift+T`）

## 创建你的第一个模板

### 第一步：创建模板文件夹

在文件列表中创建一个 `模板` 文件夹。

### 第二步：创建日记模板

在 `模板` 文件夹中新建 `日记模板.md`，写入：

```markdown
# {{date:YYYY-MM-DD}} {{title}}

## 📌 今日要点

- 

## ✅ 待办事项

- [ ] 

## 📝 笔记

- 

## 💭 思考

- 

## 📖 阅读/输入

- 

## 明日计划

- 
```

### 第三步：使用模板

1. 新建一篇笔记
2. 按 `Ctrl+Shift+T`（或通过命令面板选择"插入模板"）
3. 选择"日记模板"

## 模板中的占位变量

Obsidian 的模板支持动态变量，插入时会自动替换为实际内容：

| 变量 | 替换为 |
|-|-|
| `{{title}}` | 当前笔记的标题 |
| `{{date}}` | 当前日期（默认格式：YYYY-MM-DD） |
| `{{date:YYYY-MM-DD}}` | 自定义日期格式 |
| `{{time}}` | 当前时间（默认格式：HH:mm） |
| `{{time:HH:mm:ss}}` | 自定义时间格式 |

### 常用日期格式

```
{{date:YYYY-MM-DD}}     → 2026-06-17
{{date:YYYY年MM月DD日}}  → 2026年06月17日
{{date:MM/DD}}          → 06/17
{{date:dddd}}           → 星期四（星期几）
```

## 进阶：Template 插件 vs Templater 插件

| 特性 | 内置模板插件 | Templater 社区插件 |
|-|-|-|
| 安装方式 | 内置，直接启用 | 社区插件市场安装 |
| 变量 | 基础（日期、标题） | 强大（文件路径、剪贴板、公式计算） |
| 自定义函数 | 不支持 | 支持 JavaScript 函数 |
| 学习成本 | 零 | 需要一些学习 |
| 推荐场景 | 入门用户 | 进阶用户 |

> **建议**：刚开始用内置模板插件就够了。用熟练了之后再考虑装 Templater。
