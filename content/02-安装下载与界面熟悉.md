## 下载 Obsidian

### 官网下载

访问 [obsidian.md](https://obsidian.md)，点击首页的"下载"按钮，选择对应操作系统的版本：

- **Windows**：下载 `.exe` 安装包
- **macOS**：下载 `.dmg` 文件（Apple Silicon 芯片选 ARM 版）
- **Linux**：下载 AppImage 或通过包管理器安装
- **移动端**：iOS / Android 在应用商店搜索"Obsidian"

### 版本选择

无需付费下载——Obsidian 个人使用完全免费。付费版本（Catalyst 和 Commercial）提供的是"提前体验"等额外权益，核心功能全部免费。

## 安装步骤（Windows 为例）

1. 双击下载的 `Obsidian.xxx.exe` 文件
2. 按提示完成安装
3. 启动 Obsidian，首次打开会看到启动界面
4. 选择"创建新库"，命名为"我的知识库"
5. 选择一个本地路径存放（建议放在 `D:\` 或文档目录下）
6. 点击"创建"进入主界面

## 界面布局

Obsidian 的界面主要分为三个区域：

```
┌─────────────────────────────────────────────┐
│  标题栏（标题 + 功能按钮）                     │
├──────────┬──────────────────────┬─────────────┤
│          │                      │             │
│  左侧栏  │    编辑区            │  右侧栏     │
│          │                      │             │
│  文件列表 │  你写笔记的地方       │  反向链接   │
│          │                      │  图谱       │
│  搜索     │                      │  大纲       │
│          │                      │             │
│  书签     │                      │             │
│          │                      │             │
├──────────┴──────────────────────┴─────────────┤
│  状态栏（插件状态、字数统计等）                   │
└─────────────────────────────────────────────┘
```

### 左侧栏（核心功能区）

| 功能 | 说明 |
|-|-|
| **文件列表** | 显示 Vault 里所有文件和文件夹 |
| **搜索** | 全局搜索笔记内容（快捷键 `Ctrl+Shift+F`） |
| **书签** | 收藏重要笔记快速访问 |
| **切换功能区** | 在不同面板间切换 |

### 编辑区（核心工作区）

- **编辑模式**：直接写 Markdown 内容
- **阅读模式**：渲染后的预览效果
- **分栏编辑**：可以同时打开多个笔记

### 右侧栏（信息面板）

| 功能 | 说明 |
|-|-|
| **反向链接** | 显示有哪些笔记链接了当前笔记 |
| **图谱** | 当前笔记的局部知识网络 |
| **大纲** | 根据标题自动生成目录结构 |
| **标签面板** | 查看所有标签及其使用情况 |

## 常用快捷键

| 操作 | Windows | macOS |
|-|-|-|
| 新建笔记 | `Ctrl+N` | `Cmd+N` |
| 搜索 | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| 打开快速切换 | `Ctrl+O` | `Cmd+O` |
| 切换编辑/预览 | `Ctrl+E` | `Cmd+E` |
| 插入链接 | `Ctrl+K` | `Cmd+K` |
| 加粗 | `Ctrl+B` | `Cmd+B` |
| 斜体 | `Ctrl+I` | `Cmd+I` |

## 基础设置建议

打开设置（左下角齿轮图标），建议先调整这几项：

### 编辑器

- **严格换行**：关闭（这样单换行不会产生新段落，更符合习惯）
- **拼写检查**：按需开启
- **显示行号**：写代码时建议开启

### 文件与链接

- **自动更新内部链接**：开启（重命名文件时自动更新所有引用）
- **检测所有类型的文件**：保持默认
- **默认位置存储附件**：建议选择"当前文件夹下的指定子文件夹"，统一管理图片

### 外观

- **主题**：内置浅色/深色主题，也可以从社区安装更多主题
- **字体**：可以自定义编辑字体（推荐等宽字体写代码）
