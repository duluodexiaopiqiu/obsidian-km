# 5. 创建第一篇笔记 & 双向链接（核心功能）

## 创建你的第一篇笔记

### 方式一：文件列表新建

1. 点击左侧文件列表顶部的"新建笔记"图标（或按 `Ctrl+N`）
2. 输入笔记标题，比如"我的第一篇 Obsidian 笔记"
3. 在编辑区写入内容

### 方式二：快速切换

1. 按 `Ctrl+O` 打开快速切换
2. 输入新笔记的标题
3. 回车即可创建并打开

## 什么是双向链接？

**双向链接是 Obsidian 的灵魂功能。**

传统的链接是单向的——你在笔记A里放一个链接可以跳到笔记B，但笔记B并不知道笔记A链接了它。

双向链接则相反：

- 你在笔记A中用 `[[笔记B]]` 创建链接
- 笔记B中会自动显示"被笔记A引用"

这就像给你的知识网络装了一个**反向追踪器**。

## 如何创建双向链接

### 基本语法

```
[[笔记标题]]
```

在 Obsidian 中输入 `[[` 就会弹出搜索框，搜索并选择你要链接的笔记。

### 自定义显示文本

```
[[笔记标题|显示的文字]]
```

例如 `[[Obsidian介绍|了解Obsidian]]` 会显示为"了解Obsidian"，但链接到的是"Obsidian介绍"这篇笔记。

### 链接到标题

```
[[笔记标题#章节标题]]
```

可以精确链接到某篇笔记的特定章节。

### 链接到块

```
[[笔记标题#^块ID]]
```

链接到某个具体段落（块引用），这在做精细化笔记时非常有用。

## 反向链接在哪里看？

打开一篇笔记，在**右侧栏的"反向链接"面板**中，你会看到：

1. **已链接的引用** —— 用 `[[笔记名]]` 语法明确引用了当前笔记的笔记
2. **未链接的引用** —— 提到了当前笔记的名字但没有用链接语法的笔记（Obsidian 会自动识别）

这就形成了真正的双向连接。

## 一个实战例子

假设你在写三篇笔记：

**笔记1：Markdown语法.md**

```markdown
# Markdown 语法
Markdown 是一种轻量级标记语言，常用于 [[Obsidian]] 笔记中。
```

**笔记2：Obsidian介绍.md**

```markdown
# Obsidian 介绍
Obsidian 是一款基于 [[Markdown]] 的知识管理工具。
```

**笔记3：双向链接.md**

```markdown
# 双向链接
[[Obsidian]] 中的 [[Markdown]] 笔记可以通过 [[双向链接]] 连接起来。
```

当你打开"Markdown语法.md"时，右侧反向链接面板会显示：

- Obsidian介绍.md 链接了它
- 双向链接.md 也链接了它

你的知识就这样被**织成了一张网**。

## 为什么双向链接如此重要？

### 1. 发现知识关联

你写笔记时可能没意识到某些概念是相关的，双向链接帮你**自动发现**这些关联。

### 2. 上下文找回

当你看到一篇旧笔记的反向链接时，你会想起"哦，原来我之前在那个项目里用过这个知识点"。

### 3. 构建知识网络

每一篇笔记不再是信息孤岛，而是知识网络中的一个节点。链接越多，网络越强。

## 小练习

1. 创建三篇笔记，分别写三个不同的主题
2. 用 `[[笔记名]]` 让它们互相链接
3. 打开其中一篇，观察右侧的反向链接面板
4. 试试 `[[笔记名|自定义文字]]` 的用法
