# 9.1 必备插件推荐

下面按使用场景推荐几款最值得安装的插件，新手先从**基础篇**开始。

---

## 基础篇（必装）

### 1. Chinese Calendar（农历日期）
- **用途**：在日记中插入农历日期、节气
- **推荐理由**：中文用户写日记的标配
- **安装**：社区插件搜索"Chinese Calendar"

### 2. Paste URL into selection（粘贴 URL 到选中文字）
- **用途**：选中文字后粘贴链接，自动将链接应用到选中的文字上
- **推荐理由**：写笔记时频繁用到，省去手动 Markdown 链接的麻烦
- **安装**：社区插件搜索"Paste URL into selection"

### 3. Image Toolkit（图片工具）
- **用途**：点击图片放大查看、旋转、拖拽
- **推荐理由**：Obsidian 本身图片查看功能较弱，这个插件补齐短板
- **安装**：社区插件搜索"Image Toolkit"

### 4. Editing Toolbar（编辑工具栏）
- **用途**：在编辑区上方添加一个格式化工具栏（加粗、斜体、标题等按钮）
- **推荐理由**：不习惯快捷键的用户用鼠标点即可操作
- **安装**：社区插件搜索"Editing Toolbar"

---

## 效率篇（按需安装）

### 5. Calendar（日历视图）
- **用途**：在侧边栏显示日历，点击日期快速创建/跳转日记
- **推荐理由**：写日记用户的必备
- **安装**：社区插件搜索"Calendar"

### 6. Dataview（数据查询）
- **用途**：像数据库一样查询你的笔记，自动生成汇总表格
- **推荐理由**：Obsidian 最强大的插件之一，学会之后效率翻倍

**示例：** 显示所有 #进行中 的项目
````dataview
TABLE 状态, 开始日期, 截止日期
FROM #进行中
SORT 截止日期 ASC
````

- **安装**：社区插件搜索"Dataview"
- **学习成本**：中等，建议熟悉基础操作后再学

### 7. Periodic Notes（周期性笔记）
- **用途**：自动创建每日/每周/每月笔记
- **推荐理由**：配合日记模板和 Calendar 插件，形成完整的日记体系
- **安装**：社区插件搜索"Periodic Notes"

### 8. Quick Add（快速添加）
- **用途**：通过命令面板快速创建特定类型的笔记
- **推荐理由**：减少重复操作，一键创建日记、读书笔记等
- **安装**：社区插件搜索"Quick Add"

---

## 美化篇（纯视觉）

### 9. Blue Topaz（主题）
- **用途**：一款美观的中文主题，优化了中文排版
- **推荐理由**：如果不喜欢 Obsidian 默认外观，换这个主题准没错
- **安装**：设置 → 外观 → 主题 → 管理 → 搜索"Blue Topaz"

### 10. Style Settings（样式设置）
- **用途**：配合主题使用，精细化调整界面样式
- **推荐理由**：许多主题依赖这个插件来提供自定义选项
- **安装**：社区插件搜索"Style Settings"

---

## 进阶篇（熟悉后再说）

| 插件 | 用途 | 学习成本 |
|------|------|---------|
| **Templater** | 高级模板（支持 JavaScript） | 较高 |
| **Obsidian Git** | Git 自动同步 | 中（需懂 Git 基础） |
| **Kanban** | 看板视图 | 低 |
| **Excalidraw** | 手绘风格白板/画图 | 低 |
| **Mind Map** | 思维导图 | 低 |

---

## 插件安装清单（新手推荐顺序）

```
第1步：Editing Toolbar + Paste URL into selection（立刻提升编辑体验）
第2步：Calendar + Periodic Notes（建立日记系统）
第3步：Image Toolkit（图片查看增强）
第4步：Blue Topaz 主题（美化界面）
第5步：Dataview（查询和汇总笔记）
第6步：Quick Add（进一步提高效率）
```

> **提醒**：以上插件不需要一次性全部安装。先装 2-3 个用熟了，再逐步增加。

## 小练习

1. 关闭安全模式，安装 Editing Toolbar 插件
2. 安装 Calendar 插件，看看日历视图
3. 安装 Image Toolkit 插件，在笔记中插入一张图片试试点击放大
4. 去外观设置中浏览一下社区主题，换一个喜欢的
